## Como usar

1. Abra `SlideOps_Lite.html` no Chrome ou Edge
2. Na aba "Slides", selecione GRADE e digite o intervalo (ex: 1-45)
3. Clique em "Criar/Adicionar slides do intervalo"

## Sync entre computadores

- Backup > Baixar JSON (exporta)
- Backup > Importar (no outro PC)

Ou use "Abrir JSON existente" apontando para um arquivo no OneDrive/Dropbox.
