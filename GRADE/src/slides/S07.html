DOMÍNIO GRADE — INDIREÇÃO

Indireção: a evidência "serve" para este paciente?

Quando P / I / C / O do estudo diferem da pergunta clínica, a ponte cresce — e a certeza pode cair.
CHECKLIST PICO — APLICABILIDADE Core GRADE 5
P
População
Idade, risco basal, comorbidades, cenário clínico (sintomático vs assintomático).
I
Intervenção
Dose, intensidade, adesão, cointervenções (ex.: estatina ± ezetimiba).
C
Comparador
Placebo vs padrão de cuidado; diferença real entre as estratégias comparadas.
O
Desfecho
MACE vs LDL (substituto), duração de follow-up, definição clínica do evento.
Desfecho substituto: LDL validado como substituto para estatinas (MACE ∝ redução de LDL), mas a cadeia causal varia por classe — sempre verificar se o desfecho do estudo é o que importa para o paciente.
Quanto rebaixar
−1 Moderada — diferenças em P/I/C/O com impacto provável na magnitude do efeito.
−2 Grave — diferenças sérias ou desfecho substituto com incerteza relevante.
Onde isso aparece no CAC
• Exame anatômico (sintomático) ≠ prevenção primária com CAC
• LDL (substituto) ≠ MACE (desfecho clínico duro)
• Coortes → decisão terapêutica: confusão residual
• Dados de populações americanas (MESA) aplicados a contexto local
Upgrade possível (+1)
Efeito grande ("magnified effect"): diferença muito expressiva e biologicamente plausível.
Confundimento plausível tenderia a reduzir o efeito observado, não a criá-lo.
Essência: indireção não é "erro" — é transferência . O GRADE distingue validade interna (qualidade do estudo) de aplicabilidade (serve para este paciente?).
Na prática: pergunte — "O estudo testou o que eu faria, no paciente que eu trato, medindo o que importa?"
Regra mental: "Quanto maior a ponte entre a evidência e a pergunta clínica, menor a certeza."
