DXA da coluna: artefatos + regra ISCD

Criterios para exclusao vertebral e uso adequado de vertebras disponiveis

Artefatos que afetam DMO

* Osteófitos -> -> DMO aparente

* Calcificação aórtica -> -> DMO

* Fratura / Material cirúrgico

* Escoliose grave / Costelas sobrepostas

[!] Regra ISCD em 4 passos

* Comece L1--L4 (4 vertebras)

* Exclua se artefato OU >1,0 T-score com adjacente

* Use 3 se necessário; use 2 vertebras se 3 nao forem avaliáveis

1 vertebra -> nao usar para diagnostico
