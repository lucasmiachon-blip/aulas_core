Aplicável a GC oral >=3 meses

Indicações de Tratamento (>=50 anos)

R7 Fratura recente (<2 anos)

R8 GC >= 7,5 mg/d prednisona

R9 Idade >= 70 anos (independente T-score)

R10 T-score <= −1,5

R11 FRAX ajustado > limiar nacional

R12/R17 Pré-menopausa / Homens < 50 anos

Z-score <= −2 e/ou fratura → encaminhar especialista

R16: Início Imediato (sem esperar DXA)

Se GC >= 7,5 mg/d e sem fratura recente:

Iniciar ALN imediatamente

1ª Linha por Categoria de Risco

RISCO MÉDIO

Sem fratura recente (<2 anos)

→ ALN ou RIS (BP oral)

RISCO ALTO

Fratura recente e/ou fx vertebral >= Genant 2

→ ZOL ou DEN

RISCO MUITO ALTO

>=70a + fx recente quadril/pelve e/ou fx vertebral >=G2

→ TERI (teriparatida)

R18: Terapia Sequencial Obrigatória

Após DEN: BP para evitar rebote

Após TERI: BP ou DEN para consolidação

ALN=alendronato | ZOL=zoledronato | DEN=denosumabe | TERI=teriparatida
